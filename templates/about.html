ABOUT ShareInEase application


The main use of this project is to share any doc or file by using QR code to circulate the file in ease. Our project is most probably made upon Python, Django (some python-libraries like Django, datetime, itertools etc.)  and HTML, CSS, JavaScript for the backend and frontend respectively. Here we basically focused not only on file sharing but few other features like user authentication, faculty dashboard to have secure and better user experience.
